# 🎤 KTV AI Studio - 人工智慧伴唱帶生成器

基於 Python 與 Tkinter 打造的本地端全能影音處理 GUI 工具。深度整合 `yt-dlp`、`FFmpeg` 與頂級 AI 語音分離模型 (UVR/Demucs)，為您提供自動化的 YouTube 批次下載、AI 人聲分離與 KTV 伴唱帶生成體驗。

## ✨ 四大強大功能

1. **📺 YouTube 批次轉 MKV **
   - 支援一次貼上多行 YouTube 網址！程式會自動依序下載影片，透過 AI 提取人聲與伴奏，最後合成具備「導唱 (人聲+伴奏) / 伴唱 (純伴奏)」雙音軌切換功能的 KTV 專用 MKV 檔。
2. **📥 高品質 YouTube 批次下載**
   - 同樣支援多筆網址同時導入。支援 MP3 與 MP4 雙格式下載，預設提取 320kbps 頂級音質。
3. **🎬 本地影片批次轉檔**
   - 讓舊影片翻新！支援多檔案與資料夾導入，自動擷取音訊並分離，快速製作成 KTV 伴唱格式。
4. **🎵 專業級音檔批量分離**
   - 支援 MP3、WAV、FLAC 等格式，批次處理大量音軌，提煉純人聲與伴奏。

## 🚀 特色：一鍵自動部署環境
不需具備複雜的程式背景，點擊介面上的「**一鍵修復/初始化環境**」，程式將自動下載並配置：
- 獨立的可攜式 Python 核心
- FFmpeg 影音處理引擎
- PyTorch GPU 加速庫與 ONNX Runtime
- 多款主流 AI 人聲分離模型 (支援 MDX 與 Demucs 架構)

## 🛠️ 使用說明

### 系統需求
- **作業系統**: Windows 10 / 11 (64-bit)
- **硬體推薦**: 建議具備 NVIDIA 獨立顯示卡，以啟用 CUDA 獲得最高 10 倍以上的 AI 處理加速。

### 執行步驟
1. 確保已安裝基本 Python 環境。
2. 執行主程式：
   ```bash
   python ktv.py
   ```
3. **初次啟動**：請務必先點擊介面上的「一鍵修復/初始化環境」，等待程式將所需的 AI 引擎與模型下載完畢。
4. **開始使用**：
   - 選擇上方對應的功能分頁。
   - 點擊「🔍 檢測 GPU 環境」確認加速已開啟。
   - **網址輸入**：支援一行一個網址。若您已複製多個網址，直接**點擊輸入框即可自動分行貼上**。
   - 設定「輸出目錄」。
   - 點擊下方綠色或粉紅色大按鈕即可開始批次執行任務！

## 📂 資料夾結構說明 (由程式自動生成)
- `engine_ffmpeg/`: 存放 FFmpeg 執行檔。
- `runtime_python/`: 存放內建 Python 與 pip 環境。
- `ai_libraries/`: 存放 PyTorch、ONNX 等 AI 依賴套件。
- `ai_models/`: 存放音訊分離模型 (如 `.onnx`, `.yaml`)。
- `output/`: 預設的檔案輸出目錄。
